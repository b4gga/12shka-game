# Грибок против Марио 🎮

Простая аркадная HTML5-игра: ты — грибок, который прыгает на Марио сверху, зарабатывая очки.

## Как запустить

- **Вариант 1 (проще всего)**:
  1. Открой папку с проектом `c:\Game of Miron`.
  2. Дважды кликни по `index.html` — игра откроется в браузере.

- **Вариант 2 (через npm-скрипт в Windows)**:
  1. Открой PowerShell в папке `c:\Game of Miron`.
  2. Выполни команду:
     ```bash
     npm run start
     ```
  3. Откроется браузер с игрой.

## Управление

- **← / → или A / D** — движение влево/вправо  
- **Пробел / W / ↑** — прыжок  
- **R** — быстрое перезапуск игры при поражении

## Правила

- Прыгай **на Марио сверху**, чтобы его раздавить и получить очки.
- Если соприкоснёшься с ним **сбоку или снизу** — теряешь жизнь.
- Если упадёшь в бездну или далеко за экран — тоже минус жизнь.
- Чем больше очков, тем **выше уровень** и **быстрее враги**.

## Общие рекорды (для всех по ссылке)

По умолчанию рекорды хранятся только в браузере. Чтобы таблица рекордов была общей для всех игроков:

1. Создай проект на [Firebase Console](https://console.firebase.google.com).
2. Включи **Realtime Database** (не Firestore).
3. В разделе «Правила» укажи:
   ```json
   {
     "rules": {
       "mushroomGame": {
         ".read": true,
         ".write": true
       }
     }
   }
   ```
4. В разделе «Настройки проекта» → «Веб-приложение» скопируй конфиг.
5. Вставь его в `firebase-config.js`, заменив `YOUR_API_KEY` и остальные значения.

Удачи, боевой грибок!
